# プリセット画像の設定方法

募集投稿のアイキャッチ画像として使用できるプリセット画像を配置するディレクトリです。

## 必要な画像ファイル

以下の8つの画像ファイルを配置してください：

1. `business.jpg` - ビジネス関連
2. `agriculture.jpg` - 農業関連
3. `tourism.jpg` - 観光関連
4. `food.jpg` - 飲食関連
5. `craft.jpg` - 工芸関連
6. `nature.jpg` - 自然関連
7. `community.jpg` - 地域活動関連
8. `technology.jpg` - IT・技術関連

## 推奨画像サイズ

- **横**: 1200px
- **縦**: 630px
- **アスペクト比**: 16:9
- **フォーマット**: JPEG
- **ファイルサイズ**: 500KB以下推奨

## 画像の入手方法

### O-Dan（推奨）
https://o-dan.net/ja/

日本語で検索できる無料写真素材サイト。商用利用可能な画像を複数のサイトから一括検索できます。

**検索キーワード例：**
- ビジネス: 「オフィス」「会議」「ビジネスマン」
- 農業: 「田んぼ」「畑」「農作業」
- 観光: 「日本 観光地」「温泉」「旅館」
- 飲食: 「レストラン」「料理」「カフェ」
- 工芸: 「伝統工芸」「陶芸」「手作り」
- 自然: 「山」「海」「森」
- 地域活動: 「祭り」「地域イベント」「ボランティア」
- IT・技術: 「パソコン」「プログラミング」「テクノロジー」

### その他の無料画像サイト

- **Unsplash**: https://unsplash.com/
  - 高品質な写真が豊富
  - 英語での検索が必要

- **Pixabay**: https://pixabay.com/ja/
  - 日本語対応
  - 写真・イラスト・動画が豊富

- **Pexels**: https://www.pexels.com/ja-jp/
  - 日本語対応
  - 高品質な写真が豊富

## 画像のダウンロードと配置手順

1. 上記のサイトで適切な画像を検索
2. 「無料ダウンロード」または「Download」をクリック
3. サイズは「Large」または「1920×1080」程度を選択
4. ダウンロードした画像を適切なファイル名にリネーム
5. このディレクトリ（`public/images/presets/`）に配置

## ライセンスについて

- 商用利用可能な画像を選択してください
- クレジット表記が不要な画像を推奨します
- ダウンロード時にライセンス条件を必ず確認してください

## 画像が未配置の場合

画像ファイルが配置されていない場合、プレースホルダー（カテゴリ名のみ）が表示されます。
実際の運用前に、必ず8つすべての画像を配置してください。
