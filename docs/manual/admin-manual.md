## 管理者パネル（/admin）

管理者は、ユーザー管理・応募管理・操作ログの閲覧など、システム全体を管理できます。

---

### 1. 管理者パネルへのログイン

**URL:** `https://サイトURL/admin/login`

#### 手順 1-1: ログイン画面を開く

ブラウザで管理パネルのURLにアクセスすると、ログイン画面が表示されます。

![ログイン画面](screenshots/admin/01_login_empty.png)

#### 手順 1-2: メールアドレスを入力する

赤枠のメールアドレス欄に、管理者アカウントのメールアドレスを入力します。

![メールアドレス入力欄](screenshots/admin/02_login_email_highlight.png)

#### 手順 1-3: パスワードを入力し、ログインボタンを押す

パスワードを入力し、赤枠の「ログイン」ボタンをクリックします。

![ログインボタン](screenshots/admin/03_login_filled.png)

---

### 2. ダッシュボード

ログインすると、ダッシュボードが表示されます。左側のサイドバーから各機能にアクセスできます。

![ダッシュボード](screenshots/admin/04_dashboard.png)

**サイドバーのメニュー:**

| メニュー | 機能 |
|---|---|
| ダッシュボード | トップページ |
| 応募管理 | 全応募の一覧・詳細・編集・CSVダウンロード |
| Filament Shield > ロール | ロール・権限の管理 |
| システム管理 > ユーザー管理 | 全ユーザーの一覧・作成・編集・削除 |
| システム管理 > 操作ログ | 操作履歴の閲覧 |

---

### 3. ユーザー管理

#### 手順 3-1: ユーザー管理を開く

サイドバーの赤枠「ユーザー管理」をクリックします。

![サイドバー ユーザー管理](screenshots/admin/05_sidebar_users_highlight.png)

#### 手順 3-2: ユーザー一覧

全ユーザーの一覧が表示されます。氏名・メールアドレス・ロールで検索・フィルタリングが可能です。

![ユーザー一覧](screenshots/admin/06_users_list.png)

**一覧の項目:**

| 項目 | 説明 |
|---|---|
| 氏名 | ユーザーの名前 |
| メールアドレス | ログイン用のメールアドレス |
| ロール | ワーカー / 企業 / 管理者 / 役所 |
| Spatieロール | システム内部のロール（super_admin, municipal, worker, company） |
| 登録日 | アカウント作成日 |

**操作:**
- 「表示」: ユーザーの詳細を閲覧
- 「編集」: ユーザー情報を変更
- 検索欄: 氏名・メールアドレスで検索
- フィルタアイコン: ロールでフィルタリング

#### 手順 3-3: ユーザーを新規作成する

赤枠の「作成」ボタンをクリックします。

![作成ボタン](screenshots/admin/07_users_new_button_highlight.png)

ユーザー作成フォームが表示されます。必要な情報を入力して「作成」ボタンを押します。

![ユーザー作成フォーム](screenshots/admin/08_users_create_form.png)

**入力項目:**

| 項目 | 必須 | 説明 |
|---|:---:|---|
| 氏名 | ○ | ユーザーの名前 |
| メールアドレス | ○ | ログイン用（重複不可） |
| パスワード | ○ | 8文字以上 |
| ロール | ○ | ワーカー / 企業 / 管理者 / 役所 から選択 |
| Spatieロール | — | 権限管理用ロール（複数選択可） |

#### 手順 3-4: ユーザー詳細を表示する

一覧画面で赤枠の行をクリック、または「表示」リンクをクリックします。

![ユーザー行ハイライト](screenshots/admin/09_users_row_highlight.png)

ユーザーの詳細情報が表示されます。

![ユーザー詳細](screenshots/admin/10_users_view.png)

---

### 4. 応募管理

#### 手順 4-1: 応募管理を開く

サイドバーの赤枠「応募管理」をクリックします。

![サイドバー 応募管理](screenshots/admin/13_sidebar_applications_highlight.png)

#### 手順 4-2: 応募一覧

全応募データの一覧が表示されます。

![応募一覧](screenshots/admin/14_applications_list.png)

**一覧の項目:**

| 項目 | 説明 |
|---|---|
| 求人タイトル | 応募先の募集名 |
| 企業名 | 募集を投稿したホスト名 |
| 応募者名 | 応募したひらいず民の名前 |
| メールアドレス | 応募者のメールアドレス |
| ステータス | 応募中 / 承認 / 不承認 |
| 応募日 | 応募した日付 |
| 判定日 | 承認・不承認の判定日 |

#### 手順 4-3: CSVダウンロード

赤枠の「CSVダウンロード」ボタンをクリックすると、応募データをCSVファイルとしてダウンロードできます。

![CSVダウンロードボタン](screenshots/admin/15_applications_csv_highlight.png)

#### 手順 4-4: 応募詳細を表示する

一覧の赤枠の行をクリック、または「表示」リンクをクリックします。

![応募行ハイライト](screenshots/admin/16_applications_row_highlight.png)

応募の詳細情報が表示されます。

![応募詳細](screenshots/admin/17_applications_view.png)

---

### 5. 操作ログ

システムの操作履歴を閲覧できます。ユーザーの登録・編集・応募のステータス変更などが記録されています。

#### 手順 5-1: 操作ログを開く

サイドバーの赤枠「操作ログ」をクリックします。

![サイドバー 操作ログ](screenshots/admin/20_sidebar_logs_highlight.png)

#### 手順 5-2: 操作ログ一覧

操作ログの一覧が表示されます。

![操作ログ一覧](screenshots/admin/21_activity_logs_list.png)

#### 手順 5-3: 操作ログ詳細

一覧の赤枠の行をクリックすると、詳細が表示されます。

![操作ログ行ハイライト](screenshots/admin/22_activity_logs_row_highlight.png)

操作の詳細（変更内容・変更前後の値など）が確認できます。

![操作ログ詳細](screenshots/admin/23_activity_logs_view.png)

## 役場パネル（/municipal）

役場職員は、応募データの閲覧とCSVダウンロードのみ利用できます。データの編集・削除はできません。

---

### 6. 役場パネルへのログイン

**URL:** `https://サイトURL/municipal/login`

#### 手順 6-1: ログイン画面を開く

ブラウザで役場パネルのURLにアクセスすると、ログイン画面が表示されます。

![ログイン画面](screenshots/municipal/01_login_empty.png)

#### 手順 6-2: メールアドレスを入力する

赤枠のメールアドレス欄に、役場アカウントのメールアドレスを入力します。

![メールアドレス入力欄](screenshots/municipal/02_login_email_highlight.png)

#### 手順 6-3: パスワードを入力し、ログインボタンを押す

パスワードを入力し、赤枠の「ログイン」ボタンをクリックします。

![ログインボタン](screenshots/municipal/03_login_filled.png)

ログイン後、ダッシュボードが表示されます。

![ダッシュボード](screenshots/municipal/04_dashboard.png)

---

### 7. 応募一覧（閲覧）

#### 手順 7-1: 応募一覧を開く

サイドバーの「応募管理」をクリックします。

#### 手順 7-2: 応募一覧（読取専用）

全応募データの一覧が表示されます。**閲覧のみ**で、編集・削除はできません。

![応募一覧](screenshots/municipal/06_applications_list.png)

#### 手順 7-3: 応募詳細を表示する

赤枠の行をクリックすると、応募の詳細が表示されます。

![応募行ハイライト](screenshots/municipal/07_applications_row_highlight.png)

応募の詳細情報（応募者・募集内容・ステータスなど）が確認できます。

![応募詳細](screenshots/municipal/08_applications_view.png)

---

### 8. CSVダウンロード

応募データをCSVファイルとしてダウンロードできます。期間を指定してフィルタリングが可能です。

#### 手順 8-1: CSVダウンロード画面を開く

サイドバーの赤枠「CSVダウンロード」をクリックします。

![サイドバー CSV](screenshots/municipal/09_sidebar_csv_highlight.png)

#### 手順 8-2: 期間を指定する（任意）

開始日・終了日を入力して、ダウンロード対象の期間を指定できます。未指定の場合は全期間が対象です。

![CSVダウンロード画面](screenshots/municipal/10_csv_download_page.png)

#### 手順 8-3: ダウンロードを実行する

赤枠の「CSVダウンロード」ボタンをクリックすると、CSVファイルがダウンロードされます。

![CSVダウンロードボタン](screenshots/municipal/11_csv_download_button_highlight.png)

---

### 9. 操作ログ（閲覧）

システムの操作履歴を閲覧できます。**読取専用**です。

#### 手順 9-1: 操作ログを開く

サイドバーの赤枠「操作ログ」をクリックします。

![サイドバー 操作ログ](screenshots/municipal/12_sidebar_logs_highlight.png)

#### 手順 9-2: 操作ログ一覧

操作ログの一覧が表示されます。

![操作ログ一覧](screenshots/municipal/13_activity_logs_list.png)

#### 手順 9-3: 操作ログ詳細

赤枠の行をクリックすると、操作の詳細が表示されます。

![操作ログ行ハイライト](screenshots/municipal/14_activity_logs_row_highlight.png)

操作の詳細（変更内容・変更前後の値など）が確認できます。

![操作ログ詳細](screenshots/municipal/15_activity_logs_view.png)

## 補足

### アカウント情報

| パネル | URL | 説明 |
|---|---|---|
| 管理者パネル | `/admin` | ユーザー管理、応募管理、ロール管理、操作ログ |
| 役場パネル | `/municipal` | 応募閲覧（読取専用）、CSVダウンロード、操作ログ閲覧 |

### 権限の違い

| 操作 | 管理者 | 役場 |
|---|:---:|:---:|
| ユーザーの作成・編集・削除 | ○ | - |
| 応募の閲覧 | ○ | ○ |
| 応募のステータス変更 | ○ | - |
| CSVダウンロード | ○ | ○ |
| 操作ログの閲覧 | ○ | ○ |
| ロール・権限の管理 | ○ | - |

### スクリーンショットの再撮影

画面の変更があった場合は、以下のコマンドでスクリーンショットを再撮影できます。

```bash
# Docker Compose が起動していること
node docs/manual/capture.mjs
```

撮影されたスクリーンショットは `docs/manual/screenshots/` に保存されます。
